Conócenos

¡Bienvenido a nuestro espacio! Somos un equipo de seis estudiantes entusiastas del curso Codo a Codo 4.0 de Python Fullstack, unidos por nuestra pasión por la tecnología y el desarrollo web.

En nuestro proyecto, aplicamos metodologías ágiles para gestionar eficazmente nuestro desarrollo. Nos enfocamos en una API robusta construida con Flask y una base de datos MySQL alojada en PythonAnywhere. Esta arquitectura nos proporciona la flexibilidad y el rendimiento necesarios para cumplir con nuestras metas.

En el frontend, hemos utilizado HTML5, CSS3, y JavaScript para crear una interfaz amigable y atractiva. Bootstrap nos ha sido de gran ayuda para asegurar la responsividad y la consistencia visual en todos nuestros dispositivos. Todo nuestro frontend está desplegado en GitHub Pages, lo que nos permite compartir nuestro trabajo de manera accesible y eficiente.

Una de nuestras fortalezas como equipo radica en nuestra capacidad para trabajar en todas las áreas del desarrollo. Todos los miembros del equipo han contribuido tanto en el frontend como en el backend, lo que nos ha permitido aprender y crecer juntos en el proceso.
